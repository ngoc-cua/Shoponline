== Description ==

Graceful Blog is a free multi-purpose WordPress Blog theme. Its perfect for any kind of blog or website like lifestyle, fashion, travel, tech, health, fitness, beauty, food, news, magazine, blogging, personal, professional, etc. Its fully Responsive and Retina Display ready, clean, modern and minimal design. Graceful Blog is WooCommerce compatible, supports RTL(Right To Left) and is optimized for SEO.

== Frequently Asked Questions ==

= How to install Graceful Blog theme =
1. Login your WordPress dashboard.
2. Go to "Appearance" -> "Theme" and click on "Add New" button.
3. Click on "Upload Theme" button and choose the theme's zip file you have downloaded.
4. Click on "Install Now" button.
5. Activate the theme.
6. Now go to "Appearance" -> "Customize" to configure theme options.
